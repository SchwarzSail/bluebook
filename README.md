## bluebook

### 环境搭建（后端）
```shell
make env-up
```

### 本地运行
```shell
go mod tidy
make local
```

### 结束环境
```shell
make env-down
```

- dal: 数据访问层
- pkg: 通用工具

- app: 前端部分
- idl: 接口定义
